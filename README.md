# Fix My Street – Community Maintenance App (UI/UX)

Fix My Street is a mobile-first UI/UX prototype that enables citizens to report and track community maintenance issues such as potholes, garbage overflow, and streetlight problems.

## Problem Statement
Community Maintenance Solution – enabling citizens to report and track civic issues efficiently.

## What I Built
- Home screen with issue summary and recent issues
- Issue reporting with required field validation
- Confirmation dialog on submit
- Issue details screen with status timeline
- Bottom navigation (Home, Report, Profile)
- Photo upload interaction (UX-level)
